Código fonte / pré-formatado

O código document.getElementById('teste') foi escrito na linguagem JavaScript. E utilizando a tag "code" no código, foi possível formatar a fonte para que cada letra ocupasse o mesmo espaço dentro da palavra, essa fonte chama-se "fonte monoespaçada", muito utilizada para se ler códigos de programação.

A seguir será inserido um código na linguagem Python, onde será usado outra tag além da "code" para apresentar de maneira mais organizada e com maior facilidade de interpretação.

        
    num = int(input('Digite um número:'))
    if num % 2 == 0:
        print(f'O número {num} é PAR!')
    else:
        print(f'O número {num} é IMPAR!')
    print('Fim do programa.')

Citações completas

Segundo Sérgio Faraco, em seu livro, Lágrimas na chuva. Uma aventura na URSS:

Moscou, como escreveu aquele jornalista, não era nada daquilo que a gente imaginava. Era melhor, mas também podia ser pior.

Para realizar essa citação, utilizei a tag "blockquote - citação em bloco". Utilizadno esta tag, a citação fica com o formato acima, além de possibilitar a inserção do link de origem da referência atravéz do parâmetro "cite=". Visualmente, a inserção do link não altera nada na página, porém o navegador verá que a citação em quetão não é plágio.
