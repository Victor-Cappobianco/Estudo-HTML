Trabalhando com listas

Listas ordenadas

Listas ordenadas são listas númeradas onde a ordem dos fatores é importante, para facilitar a criação destas listas, usa-se a tag "ol" para a lista como um todo e a tag "li" para cada item. Por exemplo:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Ler
  6. Almoçar
  7. Discutir Dom Casmurro

Existe também a possibilidade de adicionar o parâmetro "type" na lista, onde é definido o tipo de lista que deseja criar, além do parâmetro "start", onde é possível definir o ponto de partida da lista.

Listas não ordenadas

São listas onde a ordem dos fatores não são exatamente relevantes, não tem a necessidade de númeração ou qualquer outro tipo de ordenamento, sendo apenas necessário a indentificação como uma lista.

Para criar esse tipo de lista, usa-se a tag "ul" e novamente a tag "li" para cada item da lista. Por exemplo:

  • Pão
  • Leite
  • Ovos
  • Alface
  • Bisnaguinha
  • Queijo
  • Pitu

Assim como no exemplo anterior, também possível adicionar formatação com o parânmetro "type", onde tem as opções: "disc" (padrão da tag), "circle" e "square".

Lista de definição

Lista de definição funciona como se fosse um dicionário, tendo o primeiro termo e a sua definição. Onde o primeiro termo é apresentado utilizando a tag "dt" e a descrição é apresentada pela tag "dd"

Por exemplo:

Semântica
Estudo sincrônico ou diacrônico da significação como parte dos sistemas das línguas naturais.
Num sistema linguístico, o componente do sentido das palavras e da interpretação das sentenças e dos enunciados.
