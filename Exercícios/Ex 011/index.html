Imagem dinâmica

Tente abrir esse site em vários dispositivos diferentes ou aumente e diminua o tamanho do navegador.

Para realizar este processo de imagem dinâmica, primeiro foi aberta a tag "picture", e dentro da tag foi adicionada a maior imagem com a tag "img". Para realizar o processo de troca de imagens foi utilizada outra tag, a tag "source", onde você indica o máximo de pixels onde a imagem vai aparecer (deixando com 50 pixels a mais para compensar a barra de rolagem). No caso foram usadas 3 imagens, onde primeiro precisa ser o arquivo menor, depois o médio e por fim, a imagem padrão, que é a maior.

Reproduzindo áudios

Vamos inserir um áudio no HTML 5.

Para reproduzir o áudio foi usada a tag "audio". Dentro da tag "audio" foi usada a tag "source:src" para escolher o arquivo a ser reproduzido, e inserindo mais de um formato de arquivo para áudio, com a finalidade de maior compatibilidade com os dispositivos dos usuários.
Dentro da tag "audio", foram inseridos os parâmetros:

  • Preload="metadata" - dizendo para só carregar as informações do arquivo e só baixar o áudio no momento que for reproduzi-lo:
  • Autoplay - para tocar o áudio automáticamente (só funciona com o parâmetro "auto");
  • Controls - adiciona um display para reproduzir o áudio;
  • Loop - toca o áudio em loop.
