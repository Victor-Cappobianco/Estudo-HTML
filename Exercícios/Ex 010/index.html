Usando links externos

A Tabacaria é um poema escrito por um dos pseudonimos de Fernando Pessoa (Álvaro Campos)

Para criar o HiperLink demonstrado acima foi utilizada a tag "a" (âncora). Nesta tag você insere o link completo do site que deseja enviar o usuário quando clicado nele.
Logo após o link, ainda dentro da tag "a", foram adicionados 2 parâmetro, o primeiro foi o parâmetro "target="_blank"" e o segundo foi o parâmetro "rel="_external"", esses parâmetros somados fazem com que, ao clicar no link, o usuário seja direcionado ao site atravéz de uma nova aba do navegador, mantendo meu site aberto."

Usando links internos

Os links internos são links que redirecionam o usuário para uma outra página ou local dentro do nosso site.

Clique aqui para receber um "oi".

Para criar esse HiperLink foi utilizado a mesma tag "a", porém utilizando o parâmetro "rel="next"" para indicar que está indo para a próxima página. E na página 2, foi utilizado o parâmetro "rel="prev"" para indicar que está voltando para a página anterior.

O HiperLink da página 2, além do parâmetro "target="_blank"" foi utilizado o paramêtro "rel="nofollow"", que indica que o site em questão é um site que não é de sua confiança, e que você não quer vínculo com ele.

Exemplo de link para páginas dentro de subpastas

Você também pode acessar nossa página de notícias.

Para acessar uma página que está dentro de uma subpasta é necessário colocar o nome da pasta no HiperLink. Porém, para retornar a página anterior, é necessário digitar "../" para que o HTML saiba que você quer ir para o endereço da página original.
